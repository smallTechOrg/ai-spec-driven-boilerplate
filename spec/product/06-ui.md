# UI

## UI Type

Server-rendered web dashboard (FastAPI + Jinja2 + minimal CSS). No JavaScript framework.

## Views / Screens

### Screen: Dashboard (`GET /`)

**Purpose:** Browse all discovered leads, filter by country / industry / status, export to CSV.

**Key elements:**
- Stub-mode banner (shown when LLM provider = stub)
- Filter bar: country dropdown, industry text, status dropdown
- Lead table: company name (linked to website), country, industry, headcount, why-fit summary, contacts (name/title), status badge, created date
- "New Search" button → navigates to search form
- "Export CSV" button → downloads filtered leads

**Actions available:**
- Filter / reset filters
- Click "New Search"
- Click "Export CSV"

### Screen: New Search Form (`GET /runs/new`)

**Purpose:** Enter search criteria and trigger a discovery run.

**Key elements:**
- Country selector (EU countries)
- Industry text field
- Min / max headcount fields (optional)
- "Find Leads" submit button
- Stub-mode banner

**Actions available:**
- Submit form → POST /runs → redirects to run results

### Screen: Run Results (`GET /runs/{id}`)

**Purpose:** Show leads discovered in a specific run; stream live progress while the run is executing; confirm run succeeded or display error.

**Key elements:**
- **Live progress feed** (shown while `status = running`): SSE-driven log of pipeline steps (search, enrich, contact, save). Auto-hides and reloads when the run completes.
- Run summary bar: status badge, criteria used, lead count, duration
- Lead table (same as dashboard table, with contacts column)
- "Export this run's leads" link
- "Back to dashboard" link
- Error message block (shown when status = `failed`)

## Error States

- **Run failed:** Red alert block on the run results page with the error message.
- **No leads found:** Empty-state message with a "Try different criteria" link.
- **Form validation error:** Inline error labels on required fields.
- **Stub mode:** Yellow banner at the top of every page: "⚠ Running in stub mode — results are simulated. Set LGA_GEMINI_API_KEY to use real data."
- **SSE disconnect:** If the EventSource connection drops, the page auto-reloads after 2 s.

## Tech Stack

FastAPI 0.115+ + Jinja2 3.x + plain CSS (no build step). All templates in `src/lead_gen_agent/templates/`. Static assets served from `src/lead_gen_agent/static/`.
